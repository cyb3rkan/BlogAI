C# Temel Sözdizimi

Değişkenler

Değişkenler, programlama dillerinde veri saklamak için kullanılan temel yapı taşlarıdır. Her değişkenin bir adı ve bir türü vardır. C#'ta değişkenler tanımlanırken türü belirtmek zorunludur.


// Tam sayı (integer)
int sayi = 10;

// Ondalıklı sayı (double)
double pi = 3.14;

// Metin (string)
string isim = "Ahmet";

// Karakter (char)
char ilkHarf = 'A';

// Mantıksal değer (boolean)
bool dogruMu = true;

Performans: Değişken türü, bellekte ayrılan alanı ve işlem hızını etkiler. Doğru türü seçmek önemlidir.

Kullanım Alanları: Veri saklama, matematiksel işlemler, kullanıcı girdilerini alma, program akışını kontrol etme.

Operatörler

Operatörler, değişkenler ve değerler üzerinde işlem yapmak için kullanılan sembollerdir. Aritmetik, karşılaştırma, mantıksal ve atama operatörleri gibi farklı türleri vardır.


// Aritmetik operatörler
int toplam = 5 + 3; // 8
int fark = 10 - 4;  // 6
int carpim = 6 * 2; // 12
int bolum = 15 / 3; // 5
int kalan = 16 % 5; // 1

// Karşılaştırma operatörleri
bool esitMi = (5 == 5);   // true
bool buyukMu = (10 > 5);  // true
bool kucukEsitMi = (3 <= 3); // true

// Mantıksal operatörler
bool ve = (true && false);   // false
bool veya = (true || false);  // true
bool degil = !true;         // false

Performans: Operatörlerin performansı genellikle hızlıdır, ancak karmaşık işlemler daha fazla zaman alabilir.

Kullanım Alanları: Hesaplama, karar verme, koşulları kontrol etme.

Döngüler

Döngüler, bir kod bloğunu belirli bir koşul sağlanana kadar tekrar tekrar çalıştırmak için kullanılır. C#'ta `for`, `while` ve `do-while` döngüleri bulunur.


// For döngüsü
for (int i = 0; i < 5; i++) {
    Console.WriteLine(i); // 0, 1, 2, 3, 4
}

// While döngüsü
int sayac = 0;
while (sayac < 5) {
    Console.WriteLine(sayac); // 0, 1, 2, 3, 4
    sayac++;
}

// Do-While döngüsü
int j = 0;
do {
    Console.WriteLine(j); // 0, 1, 2, 3, 4
    j++;
} while (j < 5);

Performans: Döngülerin performansı, tekrar sayısına ve döngü içindeki işlemlerin karmaşıklığına bağlıdır.

Kullanım Alanları: Tekrarlı işlemler, veri işleme, koleksiyonları tarama.

Karar Yapıları (if-else)

Karar yapıları, bir koşulun doğru veya yanlış olmasına göre farklı kod bloklarını çalıştırmak için kullanılır. C#'ta `if`, `else if` ve `else` ifadeleri bulunur.


int x = 10;

if (x > 5) {
    Console.WriteLine("x 5'ten büyüktür.");
} else if (x < 5) {
    Console.WriteLine("x 5'ten küçüktür.");
} else {
    Console.WriteLine("x 5'e eşittir.");
}

Performans: Karar yapılarının performansı genellikle hızlıdır. Ancak, çok sayıda `else if` bloğu performansı etkileyebilir.

Kullanım Alanları: Koşulları kontrol etme, program akışını yönlendirme, farklı senaryolara göre farklı işlemler yapma.

Diziler (Arrays)

Diziler, aynı türden birden fazla veriyi ardışık olarak saklamak için kullanılan veri yapılarıdır. Dizinin boyutu tanımlanırken belirtilir ve daha sonra değiştirilemez.


// Integer dizisi tanımlama
int[] sayilar = new int[5]; // 5 elemanlı bir dizi

// Değer atama
sayilar[0] = 10;
sayilar[1] = 20;
sayilar[2] = 30;
sayilar[3] = 40;
sayilar[4] = 50;

// Dizi elemanlarına erişim
Console.WriteLine(sayilar[2]); // 30

// Dizi tanımlama ve değer atama (kısa yol)
int[] notlar = { 85, 90, 78, 92, 88 };

Performans: Dizi elemanlarına erişim hızlıdır (O(1)), ancak dizi boyutu sabittir.

Kullanım Alanları: Sabit boyutlu veri kümelerini saklama, sıralama algoritmaları, matematiksel işlemler.
